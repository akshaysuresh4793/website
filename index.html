Latest Projects

Rcane

An R package which performs numeric optimization for Linear Regression models
New

Rcane performs Numeric Optimization using Batch Gradient Descent, Stochastic Gradient Descent, MiniBatch Gradient Descent and Coordinate Descent. The goal is to develop a high level abstraction of these algorithms implementing Linear Regression, Logistic Regression, Statistical Regularization and provide a loosely coupled solution to integrate with Convolutional and Deep Neural Networks.

Choosing Hyperparameters in Gradient Descent algorithms is always a challenge. Rcane provides a Web interface to choose different hyperparameters and evaluate the model performance.
